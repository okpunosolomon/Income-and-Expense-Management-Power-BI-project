## üìÅ Dataset Description  

The dataset used in this project, **Project_Monthly_Performance_2019_2024.xlsx**, contains six years of financial records covering **income and expense performance** for multiple projects and departments.  
It serves as the foundation for analyzing budget efficiency, cost trends, and income sustainability over time.  

### üßæ Data Overview  
- **Time Frame:** 2019 ‚Äì 2024  
- **Data Source:** Internal project performance tracking records (Excel-based)  
- **File Type:** `.xlsx`  
- **Size:** Approximately 1000+ transactional entries  

### üß© Key Columns  
| Column Name | Description |
|--------------|-------------|
| `Project_ID` | Unique identifier for each project or department |
| `Year` | Fiscal year of the record |
| `Month` | Month of transaction (used for time series trend analysis) |
| `Income_Actual` | Actual income recorded for that month/project |
| `Income_Budget` | Target income set for performance comparison |
| `Expense_Actual` | Actual expense recorded for that month/project |
| `Expense_Budget` | Planned or approved expense |
| `Variance` | Difference between budgeted and actual values |
| `% of Budget` | KPI metric measuring performance against target |
| `Remarks` | Qualitative notes or adjustments (where applicable) |

### ‚öôÔ∏è Data Cleaning & Preparation  
Initial cleaning was performed in **Microsoft Excel** before transformation in Power BI.  
Steps included:
- Removing duplicates and null entries  
- Normalizing numeric formats and currency signs  
- Standardizing month and year formats  
- Creating calculated fields for variance and budget ratios  
- Verifying consistency across fiscal years  

### üßÆ Data Transformation (Power Query)
After upload to Power BI, **Power Query Editor** was used for:  
- Column renaming and type correction  
- Deriving KPI columns (`Income Variance %`, `Expense Variance %`)  
- Filtering out invalid records  
- Establishing relationships with performance lookup tables  

### üéØ Purpose  
This dataset enables the analysis of:
- Income and expense trends over six fiscal years  
- Actual vs. budget performance at project level  
- Cost optimization opportunities  
- Year-over-year sustainability insights
